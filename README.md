# Touching Grass Simulator

A highly advanced productivity tool engineered to simulate the ancient and mystical act of touching grass — without ever leaving your room.

Because going outside is… optional.

# Preview Behavior

When the cursor hovers over the grass image:

Timer starts counting

Status updates in real time

After:

15 seconds → background turns brown

30 seconds → background turns silver

60 seconds → background turns gold 

When you stop hovering:

Timer resets

Status updates

Background returns to white



# Why This Exists?

Because sometimes the internet tells you to “touch grass”.

So we automated it.
